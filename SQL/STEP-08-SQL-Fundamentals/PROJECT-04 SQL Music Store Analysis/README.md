# 🎵 PROJECT 04 - SQL Music Store Analysis

## Overview

This project analyzes the Chinook Music Store database using SQL Server.

The objective was to answer real-world business questions related to customers, sales, genres, artists, tracks, and revenue performance using SQL queries.

---

## Dataset

The dataset used in this project is the **Chinook Database**, a sample digital music store database widely used for SQL practice and learning.

The database includes information about:

* Customers
* Employees
* Invoices
* Invoice Items
* Artists
* Albums
* Tracks
* Genres
* Playlists

## Business Questions

* How many customers does the store have?
* Which country has the most customers?
* Which country generates the highest revenue?
* Who is the top-spending customer?
* Which genre generates the most sales?
* Which artist sells the most tracks?
* Which tracks are purchased most frequently?

---

## Key Insights

### Customer Analysis

* The store serves **59 customers**.
* The **United States** has the largest customer base with **13 customers**.
* Canada ranks second with **8 customers**.

### Revenue Analysis

* The **United States** generated the highest revenue with **$523.06**.
* Canada ranked second with **$303.96** in revenue.
* France, Brazil, and Germany were among the top revenue-generating countries.

### Customer Spending

* **Helena Holý** was the highest-spending customer, with total purchases of **$49.62**.

### Genre Analysis

* **Rock** was the most popular genre with **835 sales**.
* Rock also generated the highest revenue, reaching **$826.65**.

### Artist Analysis

Top-selling artists:

1. Iron Maiden — 140 sales
2. U2 — 107 sales
3. Metallica — 91 sales
4. Led Zeppelin — 87 sales

### Track Analysis

Top-selling tracks included:

* Dazed and Confused
* The Trooper
* Eruption
* Hallowed Be Thy Name
